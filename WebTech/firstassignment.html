Types of webpages

1. Static web page

It is a web page which is delivered to the user as it is stored in the server

static web pages are used when the content of the webpage is no more required to be modified

it is made up of html and css

2. Dynamic web page

dynamic web pages show different information at different point of time

its information changed depending on the viewing time of the day, timezone, viewers langauge and at factors.

3. Single page application

It is a web page application 2hich loads only a single web document and updates the body content

the application will not load entire contents of the website,instead it only loads particular components that we want.

4. Multi page application

It is a web page application that has more than one page

it loads their pages every time a user clicks on different links
